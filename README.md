## Base

```
,------,------,------,------,------,------.   ,------,------,------,------,------,------.
|  `   |  Q   |  W   |  E   |  R   |  T   |   |  Y   |  U   |  I   |  O   |  P   |  '   |
|------+------+------+------+------+------|   |------+------+------+------+------+------|
|  =   |A  Cmd|S  Opt|D  Ctl|F  Sft|  G   |   |  H   |J  Sft|K  Ctl|L  Opt|;  Cmd|  -   |
|------+------+------+------+------+------|   |------+------+------+------+------+------|
|  [   |  Z   |X  Opt|  C   |  V   |  B   |   |  N   |  M   |  ,   |.  Opt|  /   |  ]   |
`------'------'--+---'--+---'--+---'------'   `------'--+---'--+---'--+---'------'------'
                 | ESC  | SPC  | TAB  |           | RET  | BSP  | F18  |
                 |media | nav  |mouse |           | sym  | num  | fun  |
                 `------'------'------'           `------'------'------'
```

Tap = symbol. Hold = modifier/layer.

Russian layout (Mac JCUKEN):

```
,------,------,------,------,------,------.   ,------,------,------,------,------,------.
|  yo  |  j   |  c   |  u   |  k   |  e   |   |  n   |  g   |  sh  |  shh |  z   |  eh  |
|------+------+------+------+------+------|   |------+------+------+------+------+------|
|      |f  Cmd|y  Opt|v  Ctl|a  Sft|  p   |   |  r   |o  Sft|l  Ctl|d  Opt|zh Cmd|      |
|------+------+------+------+------+------|   |------+------+------+------+------+------|
|  kh  |  ya  |ch Opt|  s   |  m   |  i   |   |  t   |  '   |  b   |yu Opt|  .   |  ""  |
`------'------'--+---'--+---'--+---'------'   `------'--+---'--+---'--+---'------'------'
```

kh = x, "" = tvyordyj znak

## Media — hold ESC

```
,------,------,------,------,------,------.   ,------,------,------,------,------,------.
|      | BOOT | TAP  |EXTRA | BASE |      |   | RGB  | EFF  | HUE  | SAT  | BRI  |      |
|------+------+------+------+------+------|   |------+------+------+------+------+------|
|      | Cmd  | Opt  | Ctl  | Sft  |      |   | Prev |Vol - |Vol + | Next |  EP  |      |
|------+------+------+------+------+------|   |------+------+------+------+------+------|
|      |      | ROpt | FUN  |MEDIA |      |   | BT0  | BT1  | BT2  | BT3  | OUT  |      |
`------'------'--+---'--+---'--+---'------'   `------'--+---'--+---'--+---'------'------'
                 |######|      |      |           | STOP |  PP  | MUTE |
                 `------'------'------'           `------'------'------'
```

Media vim-style: H=prev J=vol- K=vol+ L=next

BT0-BT3: tap = select, shift+tap = clear

## Flashing

1. Edit `config/blecorne.keymap`
2. Push -> GitHub Actions builds automatically
3. Download `.uf2` from Actions -> Artifacts
4. USB -> double tap reset -> drag `.uf2` to the drive
5. Flash both halves separately (`_left` and `_right`)
